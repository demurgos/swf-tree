# PushDuplicate

```
[value] → [value, value]
```

- Action Code: `0x4c`
- Stack: `1 → 2`
- SWF version: `5`

## Original documentation

### ActionPushDuplicate

ActionPushDuplicate pushes a duplicate of top of stack (the current return value) to the stack.

| Field               | Type               | Comment                        |
|---------------------|--------------------|--------------------------------|
| ActionPushDuplicate | ACTIONRECORDHEADER | ActionCode = 0x4C              |
