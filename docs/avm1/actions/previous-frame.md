- Action Code: `0x05`
- Stack: `0 → 0`

## Original documentation

### ActionPreviousFrame

ActionPreviousFrame instructs Flash Player to go to the previous frame of the
current file.

| Field           | Type               | Comment           |
|-----------------|--------------------|-------------------|
| ActionPrevFrame | ACTIONRECORDHEADER | ActionCode = 0x05 |
