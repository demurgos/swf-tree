# Pop

```
[a] → []
```

- Action Code: `0x17`
- Stack: `1 → 0`

## Original documentation

### ActionPop

ActionPop pops a value from the stack and discards it.

| Field              | Type               | Comment           |
|--------------------|--------------------|-------------------|
| ActionPop          | ACTIONRECORDHEADER | ActionCode = 0x17 |

ActionPop pops a value off the stack and discards the value.
