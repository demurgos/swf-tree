- Action Code: `0x04`

## Original documentation

### ActionNextFrame

ActionNextFrame instructs Flash Player to go to the next frame in the current file.

| Field           | Type               | Comment           |
|-----------------|--------------------|-------------------|
| ActionNextFrame | ACTIONRECORDHEADER | ActionCode = 0x04 |
